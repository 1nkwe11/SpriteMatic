# SpriteMatic Frontend

React + Vite SPA for SpriteMatic.

## Commands

```bash
npm run dev --workspace @spritematic/frontend
npm run build --workspace @spritematic/frontend
npm run lint --workspace @spritematic/frontend
```

## Environment

Copy `.env.example` to `.env`:

```bash
VITE_API_BASE_URL=http://localhost:4000/api
```

For full project setup, see the root `README.md`.
